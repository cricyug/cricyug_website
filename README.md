
# CricYug Final Platform

This is the one-shot starter repo.

## Included
- Frontend website
- Backend API
- Players page
- Teams page
- Matches page
- Admin placeholder
- JSON data
- Generator script

## Run locally
1. Open terminal
2. Go to the repo folder
3. Run:
   npm install
   npm start
4. Open:
   http://localhost:3000

## Files to edit
- backend/data/players.json
- backend/data/teams.json
- backend/data/matches.json

## Regenerate pages
python3 tools/generator.py
